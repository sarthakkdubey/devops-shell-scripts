## 🚀 Django Deployment Script

This script automates the deployment of a Django application using Docker.

### 📦 Project Used

The script clones and deploys the following open-source project:

* https://github.com/LondheShubham153/django-notes-app

### ⚙️ Features

* Clones or updates the project repository
* Installs Docker and Nginx
* Builds Docker image
* Runs container on port 8000
* Handles container restart safely

### ▶️ Usage

```bash
chmod +x django-docker-deploy.sh
./django-docker-deploy.sh
```
