# DevOps Shell Scripts

This repository contains shell scripting practice projects created while learning Linux, automation, and DevOps concepts.

The main purpose of this repository is to build hands-on understanding of how shell scripts are used in real-world DevOps workflows for automation, monitoring, deployments, and system administration tasks.

Each folder in this repository contains scripts related to a specific area of DevOps and Linux automation.
